# Chrome Extension: Legal and Compliance Page Analyzer

A Chrome extension that scans any web page for legal compliance issues including GDPR cookie consent gaps, ADA accessibility violations, missing privacy policy links, broken terms of service references, and FTC disclosure requirements. Produces a one-click compliance report with severity ratings and fix recommendations. Built for compliance officers, legal teams, and marketing agencies managing regulatory risk.

## Installation

### Chrome
1. Download and extract the ZIP
2. Go to `chrome://extensions/`
3. Enable "Developer mode"
4. Click "Load unpacked" and select the extracted folder

### Firefox
1. Download the Firefox version
2. Go to `about:debugging`
3. Click "Load Temporary Add-on"
